### Au Programme:  
- Présentation des boutons poussoirs :  
  - Comment ça marche?
  - Comment câbler?
  - Comment programmer? 
  - Mise en pratique:
    - Allumer une led.
    - Lancer un compteur.
- Présentation des signaux PWM :  
  - Qu'est ce que c'est? à quoi ça sert?
  - Comment programmer?
  - Mise en pratique:   
    - Faire varier la luminausité d'une led. 
- Présentation des Leds RGB :  
  - Qu'est ce que c'est? Comment ça marche?
  - Comment programmer?
  - Comment câbler?

    - Obtenir 3 couleurs.
